# Riksarkivet Tools Plugin for Claude Code

A Claude Code plugin with skills for using Riksarkivet MCP tools â€” HTR transcription, document viewing, and archive research workflows.

## Installation

Install via Claude Code:

```
/install-plugin AI-Riksarkivet/ra-mcp plugins/ra-mcp-tools
```

## Usage

Invoke skills by asking Claude Code to:

### HTR Transcription

- "Transcribe this handwritten document"
- "Run HTR on these page images"
- "Recognize handwritten text in this manuscript"
- "Transcribe old letters from these images"

### Document Viewer

- "Show this document page in the viewer"
- "View the original scanned image with text overlay"
