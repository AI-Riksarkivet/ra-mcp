# Archival Guide — Swedish Record Types and Administrative History

Use the Förvaltningshistorik (Swedish Administrative History) resources to understand
what types of records exist, which authorities created them, and where to find them.

## When to Use This Guide

Consult the guide resources BEFORE searching when:
- The user asks "where can I find records about X?"
- You need to understand what type of document a search result comes from
- You want to suggest related record types the user might not know about
- You need historical context about a specific authority or institution

## How to Access the Guides

1. **Get the table of contents** first:
   Read resource `riksarkivet://contents/table_of_contents`

2. **Load a specific chapter** by filename:
   Read resource `riksarkivet://guide/01_Domstolar.md`

## Common Research Scenarios

### Genealogy / Family History
Start with **07_Folkbokforing.md** (church records, husförhörslängder, birth/marriage/death books).
Cross-reference with **01_Domstolar.md** (bouppteckningar in court archives),
**06_Statskyrkan.md** (parish organization), and **46-61_Forsvaret.md** (military service rolls).

### Criminal Cases and Trials
Start with **01_Domstolar.md** (häradsrätt, rådhusrätt, hovrätt structure).
See **02_Fangelse.md** for prison records, **24_Polis.md** for police records,
and **37_Overvakning.md** for parole/probation.

### Property and Land
Start with **09_Lantmateri.md** (land survey, property registration).
See **01_Domstolar.md** for property disputes (bouppteckningar, lagfarter),
and **03_Skatt.md** for tax-based property records (mantalslängder).

### Immigration and Emigration
Start with **25_Invandring.md** and **07_Folkbokforing.md** (flyttningslängder).
For sailors, see **14_Sjofart.md**.

### Military Service
Start with **46-61_Forsvaret.md** (comprehensive military chapter covering army,
navy, air force, conscription, military hospitals, fortifications).
Cross-reference with **07_Folkbokforing.md** for garrison parishes.
