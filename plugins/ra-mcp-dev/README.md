# Riksarkivet Dev Plugin for Claude Code

A Claude Code plugin with development skills for building MCP apps with FastMCP and Svelte.

## Installation

Install via Claude Code:

```
/install-plugin AI-Riksarkivet/ra-mcp plugins/ra-mcp-dev
```

## Usage

Invoke skills by asking Claude Code to:

### Build MCP Apps

- "Create a new MCP workspace package"
- "Add a new MCP tool to the server"
- "Scaffold a new MCP module"
